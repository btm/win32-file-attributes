= Description
  Additional methods for the File class that relate to file properties
  on the Microsoft Windows operating system.

== Prerequisites
  * ffi

== Installation
  gem install win32-file-attributes

== Synopsis

  require 'win32/file/attributes

  p File.hidden?(file)
  p File.attributes(file)
